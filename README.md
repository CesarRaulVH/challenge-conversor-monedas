💱 Conversor de Monedas - Java

Aplicación de consola desarrollada en Java que permite convertir montos entre diferentes monedas utilizando tasas de cambio en tiempo real obtenidas desde una API externa.

Este proyecto fue desarrollado como parte del programa Oracle Next Education (ONE) junto a Alura Latam.

🚀 Funcionalidades

El programa permite convertir entre las siguientes monedas:

Dólar → Peso Argentino

Peso Argentino → Dólar

Dólar → Real Brasileño

Real Brasileño → Dólar

Dólar → Peso Colombiano

Peso Colombiano → Dólar

🌐 API utilizada

El proyecto utiliza la API pública:

ExchangeRate API

Esta API permite consultar tasas de cambio entre diferentes monedas mediante peticiones HTTP.
